# CFA Interactive Notebook

An interactive CFA learning notebook with chapters, search, review mode, memory cards, scenario quizzes, and concept maps.

## Local Preview

Run from this folder:

```bash
python3 -m http.server 4173
```

Open:

```text
http://localhost:4173/
```

## GitHub Pages

After pushing this repository to GitHub:

1. Open the repository on GitHub.
2. Go to `Settings`.
3. Open `Pages`.
4. Set Source to `Deploy from a branch`.
5. Select branch `main` and folder `/ root`.
6. Save.

The public notebook URL should become:

```text
https://ywu087-cell.github.io/cfa-notebook/
```
